Hello, I'm Conor!

I am a frontend web developer based in the north of England. I love to build websites and I am constantly learning new technologies to aid me in my passion. HTML and CSS are at the heart of my abilities, with JavaScript well on it's way. I am currently teaching myself React and hope to have some React projects up on here soon.

In my spare time I like to read. Sci-fi my favourite genre currently but it changes often! I'm also crazy about films, and I try to incorporate as much travel as I can into my life.

Here are some of my previous projects

Quote of the Day Generator

This quote generator pulls a quote as well as the author from the theysaidso.com API. The page itself is basic HTML with some CSS for design, the bulk of the work is done by the JavaScript.

TMDB API Chrome Extension

This extension replaces the new tab and pulls recent films from The Movie Database and using their API and places them in a container where they can be selected to display their information in the main window.

Todo List

This todo list uses JavaScript to store values entered into the input and display them in a list. It also has the ability to mark items as checked and can mark all items at once.

Survey Form

This page's main focus was organisation of elements, and playing around with divs to make the page read better and look presentable. It uses HTML and CSS to achieve this.
